# Pilot Use Case Lock (Vadodara)

## Focused use case
First-line symptom triage + shareable health pass before clinic visit.

## Target users
- Families in Vadodara region needing quick decision support
- Clinics receiving walk-in patients without prior records

## Pilot flow
1. User logs in and submits symptom triage.
2. App generates safe guidance and urgency level.
3. User generates health pass code.
4. Clinic opens `/doctor-view/<code>` and reviews profile + recent triage details.

## Success criteria
- Triage completion rate > 60%
- Share pass generation rate > 25%
- Doctor summary views recorded for real consultations
