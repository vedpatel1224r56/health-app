# સેહતસાથી દર્દી ઓનબોર્ડિંગ (ગુજરાતી)

## આ એપ શું કરે છે
- તમારા લક્ષણોના આધાર પર પ્રથમ સ્તરની આરોગ્ય માર્ગદર્શન આપે છે.
- ઘરે સંભાળ પૂરતી છે કે ક્લિનિક મુલાકાત જરૂરી છે તે નક્કી કરવામાં મદદ કરે છે.
- હેલ્થ પાસ કોડ દ્વારા ડૉક્ટર સાથે પ્રોફાઇલ અને તાજેતરનું માર્ગદર્શન શેર કરવા દે છે.

## કેવી રીતે ઉપયોગ કરવો
1. એકાઉન્ટ બનાવો અને પ્રોફાઇલ પૂર્ણ કરો.
2. સિમ્પ્ટમ ટ્રાયેજ શરૂ કરો.
3. માર્ગદર્શન અને સેફટી નોટ વાંચો.
4. ક્લિનિક જતાં પહેલા હેલ્થ પાસ જનરેટ કરો અને કોડ બતાવો.

## સેફટી નોટ
આ એપ નિદાન નથી. ઇમરજન્સીમાં તરત સારવાર લો.

## ડેટા અને પ્રાઇવસી
પ્રોફાઇલ, ટ્રાયેજ અને કન્સેન્ટ સતત કાળજી માટે લોગ થાય છે. તમે ડેટા એક્સપોર્ટ/ડિલીટ કરી શકો છો.
