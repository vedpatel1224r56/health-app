# Privacy Policy (Draft v1)

Effective date: 2026-02-12

SehatSaathi collects personal and health-related information to provide symptom triage guidance and patient profile services.

## Data collected
- Account data: name, email, encrypted password hash
- Health profile: age, sex, conditions, allergies, region
- Triage data: symptoms, severity, duration, red flags, generated guidance
- Optional uploads: symptom photos
- Operational logs: audit events, consent records, analytics events

## How data is used
- To generate guidance and improve user safety
- To maintain continuity of care via profile/history/share pass
- To satisfy security, compliance, and audit requirements

## Data sharing
- Health pass sharing happens only when a valid share code is generated by the user
- No sale of personal data

## Retention
- Data is retained until user deletion request or policy-driven archival
- Audit and analytics logs are retained for operational and compliance needs

## User rights
- Export personal data via app API
- Delete account and associated records via app API

## Security controls
- Password hashing
- Token-based authentication
- Access control and request audit logs
